变量

变量名

  • 自定义的变量名首尾不能都有双下划线
  • 受保护的变量一般用单个下划线开头
  • 私有的变量一般用两个下划线开头

链式赋值

一个值可以赋给多个变量（标签/指针）

n = age = chest = 183
print(n, age, chest)
183 183 183

交换赋值

age = 15
chest = 20
age, chest = chest, age
print(age, chest)
20 15

注释

  1. 以 # 开头，只能表示单行注释
  2. 用一对 ''' 或 """，实为多行字符串，但由于没有返回值，可以表示多行注释
"""
自定义函数

Version: 0.1
Author: 骆昊
"""
import math

def foo(x):
    """斐波那契数列"""
    return math.factorial(x)

if __name__ == "__main__":
    print(foo(5))
120

缩进与空行

用 : 和缩进表示代码块

函数之间或类的方法之间用空行分隔，表示一段新的代码的开始。类和函数入口之间也用一行空行分隔，以突出函数入口的开始。

空行与代码缩进不同，空行并不是 Python 语法的一部分。书写时不插入空行，Python 解释器运行也不会出错。但是空行的作用在于分隔两段不同功能或含义的代码，便于日后代码的维护或重构。

语句

行中语句

同一行中可以使用多条语句，语句之间使用分号 ; 分割

import sys; x = 'runoob'; sys.stdout.write(x + '\n')
runoob
7

其中 7 表示字符数：'runoob' 有 6 个字符，\n 表示一个字符，加起来 7 个字符

多行语句

反斜杠 \ 代表换行

在 [], {}, 或 () 中的多行语句，不需要使用反斜杠 \

total = 1 + \
        2 + \
        3
print(total)
